# gmatcha

## how to run the project

**install dependencies**:

```bash
npm install
```

**set up environment variables**:

```bash
cp .env.example .env
```

**run the project**:

```bash
npm run dev
```

**open the project**:

open your browser and go to [http://localhost:3000](http://localhost:3000)
